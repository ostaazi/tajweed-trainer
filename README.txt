
tajweed-trainer — v7 patch
==========================

ما الذي يتضمنه هذا التحديث؟
- محوّل الوضع (فاتح/داكن) مع حفظ التفضيل في localStorage.
- إصلاح احتساب النتيجة: السؤال غير المُجاب عنه يُحسب خطأ (لا يُسجّل صواب تلقائياً).
- عودة المنزلق الثلاثي (مع مقبضين) لتوزيع الأسئلة بين الأقسام الثلاثة ويظل ظاهراً دائماً.
- إبراز الكلمات المستهدفة بين [[...]] داخل متن السؤال بلون أخضر وحجم خط أكبر.
- صفحة نتيجة تفصيلية محسّنة مع مؤشرات ✔︎/✖︎ وأزرار طباعة/PDF.

كيفية التركيب السريع:
1) انسخ/استبدل الملفين التاليين داخل مشروع GitHub Pages لديك:
   - quiz.html
   - quiz.js  (اختياري، للاحتفاظ بالتوافق إن كان مستورداً)
2) أبقِ ملف questions_bank.json في نفس المسار (ルート أو نفس مجلد الصفحة).
3) افتح الصفحة مع باراميتر نسخة لتجاوز الكاش، مثلاً:
   https://your-domain/quiz.html?v=7

ملاحظات:
- يعتمد هذا الإصدار على هيكل بنك الأسئلة بصيغته الحالية (sections -> parts -> array of questions).
- نطاق الأجوبة الافتراضي في العرض الحالي: [إظهار/إدغام/إخفاء/إقلاب] كاختيارات راديو.
- إن كانت لديك خيارات خاصة بكل قسم، يمكن توسيع الدالة renderer بسهولة.
