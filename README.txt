Tajweedy — مشروع كامل (اختبار + تدريبات + مدقق)
1) ارفع ملفك الأساس: questions_bank.json بجوار الصفحات.
2) exercises_bank.json (مرفق هنا) هو بنك التدريبات (30 سؤال لكل حكم فرعي). يمكنك تعديله لاحقًا.
3) افتح validator.html لفحص عدم التكرار بالنص بين البنكين.
4) افتح quiz.html للاختبار (5 أسئلة لكل حكم فرعي) — النتائج تُحفظ محليًا.
5) افتح exercises.html للتدريبات (5 في كل دفعة مع زر المزيد) — النتائج تُحفظ محليًا.
